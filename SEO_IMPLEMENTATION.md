- [x] **Aucun mot interdit utilisé** : Vérifié dans tous les titles, descriptions, OG, Twitter Cards. Aucun mot de la liste interdite (solutions, services, expertise, accompagnement, méthode, processus, framework, best practices, résultats garantis, transformation, succès, performance, témoignages, case studies, ressources, newsletter).

- [x] **Aucun CTA marketing ajouté** : Tous les textes sont factuels et descriptifs. Aucun "Réservez maintenant", "Découvrez", "En savoir plus" dans les metas.

- [x] **Aucun contenu survendu** : Tous les textes reformulent fidèlement le contenu réel. Aucune promesse implicite, aucun verbe de promesse (améliorer, booster, transformer).

- [x] **SEO aligné avec le positionnement filtrant** : Les metas reflètent le positionnement "pas de méthode standard", "pas de framework plaqué", "pas de promesse marketing". Les concepts centraux (dette de décision, arbitrage, système, responsabilité) sont intégrés naturellement.

### Structure proposée pour Layout.astro

```astro
---
interface Props {
  title?: string;
  description?: string;
  ogTitle?: string;
  ogDescription?: string;
  ogType?: 'website' | 'article';
  ogImage?: string;
  ogUrl?: string;
  twitterCard?: 'summary' | 'summary_large_image';
  twitterTitle?: string;
  twitterDescription?: string;
  articleSchema?: object;
  personSchema?: object;
}

const {
  title = 'Julien Brionne — Product Lead Indépendant',
  description = 'Product Lead indépendant. J\'interviens auprès de fondateurs et Heads of Product (Series A/B) quand le système produit a grandi plus vite que sa capacité à trancher.',
  ogTitle,
  ogDescription,
  ogType = 'website',
  ogImage,
  ogUrl,
  twitterCard = 'summary',
  twitterTitle,
  twitterDescription,
  articleSchema,
  personSchema
} = Astro.props;

const siteUrl = 'https://julienbrionne.com';
const finalOgTitle = ogTitle || title;
const finalOgDescription = ogDescription || description;
const finalOgUrl = ogUrl || Astro.url.href;
const finalTwitterTitle = twitterTitle || finalOgTitle;
const finalTwitterDescription = twitterDescription || finalOgDescription;
---

<!doctype html>
<html lang="fr">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <link rel="icon" type="image/svg+xml" href="/favicon.svg" />
    <link rel="icon" href="/favicon.ico" />
    <meta name="generator" content={Astro.generator} />
    
    <!-- SEO Meta -->
    <title>{title}</title>
    <meta name="description" content={description} />
    
    <!-- Open Graph -->
    <meta property="og:title" content={finalOgTitle} />
    <meta property="og:description" content={finalOgDescription} />
    <meta property="og:type" content={ogType} />
    <meta property="og:url" content={finalOgUrl} />
    {ogImage && <meta property="og:image" content={`${siteUrl}${ogImage}`} />}
    
    <!-- Twitter Card -->
    <meta name="twitter:card" content={twitterCard} />
    <meta name="twitter:title" content={finalTwitterTitle} />
    <meta name="twitter:description" content={finalTwitterDescription} />
    {ogImage && <meta name="twitter:image" content={`${siteUrl}${ogImage}`} />}
    
    <!-- Schema.org JSON-LD -->
    {articleSchema && (
      <script type="application/ld+json" set:html={JSON.stringify(articleSchema)} />
    )}
    {personSchema && (
      <script type="application/ld+json" set:html={JSON.stringify(personSchema)} />
    )}
  </head>
  <body class="bg-[#FDFCFB] text-gray-900 antialiased relative">
    <!-- ... reste du body ... -->
  </body>
</html>
```
